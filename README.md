### Setup steps (for development)

```
$ npm install
$ ./node_modules/.bin/electron-rebuild
$ npm run watch &
$ npm run start
```

### Build steps (for production)

#### Mac OS X:


```
$ npm install
$ npm build-osx

```

#### Windows:

```
$ npm install
$ npm build-win
```
