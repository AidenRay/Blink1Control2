### Setup steps (for development)

```
$ npm install
$ npm run postinstall
$ npm run watch &
$ npm run startdev
```
Or to emulate a production run:
```
$ npm install
$ npm run postinstall
$ npm run webpack
$ npm run start
```

### Build steps (for production)

#### Mac OS X:


```
$ npm install
$ npm run postinstall
$ npm run webpack
$ npm run dist:mac

```

#### Windows:

```
$ npm install
$ npm run webpack
$ npm run dist:win
```
